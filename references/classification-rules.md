# DocMind 分类规则摘要

完整版见仓库 `docs/CLASSIFICATION_RULES.md`。

## 顶层（三选一）

| 目录 | 说明 |
|------|------|
| 生活 | 私人文件：照片、笔记、医疗、个人账单 |
| 工作 | 工作文件：合同、项目文档、会议纪要 |
| 应用快捷方式 | `.lnk` / `.url`，仅 1 级 |

## 路径模板

- 工作办公：`工作/办公/{子类}`
- 工作技术资料：`工作/技术资料/{子类}`
- 工作项目：`工作/{项目名}/{子类}`（优先 3 级，不用 `工作/项目/…`）
- 生活财务：`生活/财务/{子类}`
- 生活日常：`生活/日常/{子类}`
- 兜底：`其他/未分类`

## 增量整理

用户执行 `confirm-structure` 后，后续整理优先放入归档根下**已有**文件夹；无合适目录时再按标准新建。

## 习惯学习

首次引导可选扫描桌面/文档等（排除系统目录与 `node_modules` 等），将已有文件夹名合并进 `categories`。
